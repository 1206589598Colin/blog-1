## _Html, Css, Javascript_

> 前端 web 技术的基石：html + css + js
>
> - HTML 是我们用来构造网站内容的不同部分并定义它们的意义或目的的语言。
> - 我们可以使用 CSS 这个语言来设计和布局我们的 Web 内容，以及添加像动画一类的行为。
> - JavaScript 是用于向 Web 页面添加动态功能的脚本语言。
